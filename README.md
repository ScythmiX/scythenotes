A simple note-taking app that remembers your preferred theme, created notes, and edits. It features an ambient canvas background using plain JavaScript and the Canvas API.

The canvas is fully responsive to any screen size or window resize. It contains randomly moving particles that stay within the screen bounds. When particles come close to each other, they form a connection with a line, creating an interactive visual effect.

The whole project has a glowing theme and a modern look designed to appeal to the user's eye.

---

## Features
- Add, edit, and delete notes
- Dark / Light theme toggle
- Interactive particle canvas background
- Fully responsive design
- Notes persist using `localStorage`

---

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (ES6 modules)
- Canvas API

---

## Usage
1. Clone this repository or download it as a ZIP.
2. Open `index.html` directly in a modern browser **or**
3. Run `npm start` to serve the project with live-server.
4. Click the **+** button to add a new note
5. Edit or delete notes using the icons on each note card
6. Toggle the theme with the moon/sun button
7. Your notes are saved automatically in your browser
